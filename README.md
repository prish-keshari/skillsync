SkillSync is a full-stack AI agent platform that enables users to create, customize, and interact with intelligent AI agents through real-time chat and live video meetings.

---

## 🚀 Features

- 🤖 Custom AI agent creation with unique personas
- 📄 Retrieval-Augmented Generation (RAG) using uploaded documents
- 💬 Real-time AI chat system
- 🎥 Live video meetings with AI agents
- 🧠 Semantic search using vector embeddings
- 📝 AI-generated meeting summaries and transcripts
- 🔐 Secure authentication and protected routes
- 📊 Analytics dashboard for monitoring agent usage
- 🔑 API key generation for integrations
- 💳 Credit-based interaction system
